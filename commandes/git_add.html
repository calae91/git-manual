- git add -f

Git add -f est une fonction qui autorise l'ajout de fichiers qu'on a ignorés.

- git add -n

git add -n permet de visualiser les derniers commits de la branche où n correspond au nombre de commits que l'on cherche à visualiser en partant des plus récents
La commande est assez inutile car le git log montre les commits dans le même ordre.
